# Prakhar Kumar Portfolio

Interactive Next.js portfolio for Prakhar Kumar built around a cinematic digital India infrastructure concept.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

On this Windows machine, if plain `npm` points to a broken roaming path, run:

```powershell
node "C:\Program Files\nodejs\node_modules\npm\bin\npm-cli.js" install
node "C:\Program Files\nodejs\node_modules\npm\bin\npm-cli.js" run dev
```

## Vercel Deployment

Vercel should auto-detect this as a Next.js project.

- Framework Preset: `Next.js`
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: leave default
- Node.js Version: `20.x`

No custom `vercel.json` is required.
